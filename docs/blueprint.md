# **App Name**: Inventário Fácil

## Core Features:

- Entrada de Dados Personalizável: Permitir a entrada de dados flexível e customizável, espelhando as colunas da planilha original.
- Adicionar/Remover Linhas e Colunas: Capacidade de adicionar e remover linhas e colunas conforme necessário para expandir ou contrair o inventário.
- Cálculos Automáticos: Realizar cálculos automáticos (Estoque, Recebido, Saída) para evitar erros manuais.
- Filtros e Pesquisas: Implementar filtros para pesquisar e organizar os itens por nome, categoria ou data.
- Exportação de Dados: Permitir a exportação dos dados para formatos comuns (CSV, Excel) para compatibilidade com outros sistemas.
- Integração com o JW Hub (Generativo): Use uma ferramenta de IA para interpretar dados da planilha, resumir tendências de uso, e dar dicas baseadas nas práticas recomendadas do JW Hub.

## Style Guidelines:

- Cor primária: Azul claro (#A0CFEC), representando organização e confiança.
- Cor de fundo: Branco acinzentado (#F5F5F5) para uma aparência limpa e profissional.
- Cor de destaque: Verde (#90EE90) para indicar sucesso ou adição de novos itens.
- Fonte: 'PT Sans', uma sans-serif humanista para uma leitura clara e agradável, tanto em títulos quanto no corpo do texto.
- Ícones simples e intuitivos para representar ações (adicionar, remover, exportar).
- Layout de grade (grid) para facilitar a visualização dos dados, espelhando a estrutura da planilha.
- Transições suaves ao adicionar, remover ou editar itens para melhorar a experiência do usuário.
